💾 Backup și transfer date

Datele din Miau sunt stocate local pe telefonul tău. Folosește Export / Import JSON pentru a face backup sau a muta datele pe alt dispozitiv.

⚠️ Important: Dezinstalarea aplicației sau ștergerea datelor din browser șterge și istoricul. Fă backup periodic!

Cum faci Export (backup)

  1. Deschide Miau → tab Setări ⚙️
  2. Derulează până la secțiunea Date și backup
  3. Apasă Export JSON
  4. Dacă ai mai mulți copii configurați, apare o fereastră cu opțiunile:
    Toți — exportă datele tuturor copiilor într-un singur fișier
    Numele fiecărui copil — exportă doar datele acelui copil
  5. Telefonul descarcă fișierul miau-backup-AAAA-LL-ZZ.json
  6. Salvează-l în Google Drive, iCloud sau trimite-l pe email — ca să nu-l pierzi dacă schimbi telefonul
💡 Fișierul JSON conține tot: protocolul, istoricul tratamentelor, simptomele, stocurile, setările. Nu conține parole sau date sensibile din afara aplicației.

Cum faci Import (restaurare sau transfer)

  1. Pe telefonul nou (sau după reinstalare), deschide Miau
  2. Dacă e prima deschidere, finalizează onboarding-ul rapid (datele se vor suprascrie la import)
  3. Mergi la Setări ⚙️ → Date și backup
  4. Apasă Import JSON
  5. Selectează fișierul miau-backup-*.json salvat anterior
  6. Confirmă — aplicația încarcă toate datele din backup
⚠️ Importul suprascrie datele existente. Dacă ai tratamente noi pe telefonul curent, exportă-le mai întâi înainte să imporți un backup vechi.

Transfer între două telefoane

  1. Pe telefonul vechi: Setări → Export JSON → salvează fișierul
  2. Transferă fișierul pe telefonul nou (AirDrop, WhatsApp, email, Google Drive etc.)
  3. Pe telefonul nou: deschide Miau → Setări → Import JSON → selectează fișierul

Câte des să faci backup?

Recomandat: o dată pe lună sau înainte să schimbi telefonul. Fișierul e mic (câțiva KB) și poate fi salvat oricând.

Unde se salvează fișierul pe telefon?

iPhone: în aplicația Fișiere → Descărcări (sau iCloud Drive, dacă l-ai salvat acolo).

Android: în folderul Downloads.
