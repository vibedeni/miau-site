💾 Backup and data transfer

Miau's data is stored locally on your phone. Use Export / Import JSON to back up or move your data to another device.

⚠️ Important: Uninstalling the app or clearing browser data also deletes your history. Back up regularly!

How to Export (backup)

  1. Open Miau → Settings ⚙️ tab
  2. Scroll to the Data & backup section
  3. Tap Export JSON
  4. If you have multiple children configured, a window appears with options:
    All — exports all children's data into a single file
    Each child's name — exports only that child's data
  5. Your phone downloads the file miau-backup-YYYY-MM-DD.json
  6. Save it to Google Drive, iCloud, or email it to yourself — so you don't lose it if you change phones
💡 The JSON file contains everything: the protocol, treatment history, symptoms, supplies, and settings. It contains no passwords or sensitive data from outside the app.

How to Import (restore or transfer)

  1. On the new phone (or after reinstalling), open Miau
  2. If it's the first time opening, finish the quick onboarding (data will be overwritten on import)
  3. Go to Settings ⚙️ → Data & backup
  4. Tap Import JSON
  5. Select the previously saved miau-backup-*.json file
  6. Confirm — the app loads all the data from the backup
⚠️ Importing overwrites existing data. If you have new treatments on the current phone, export them first before importing an older backup.

Transferring between two phones

  1. On the old phone: Settings → Export JSON → save the file
  2. Transfer the file to the new phone (AirDrop, WhatsApp, email, Google Drive, etc.)
  3. On the new phone: open Miau → Settings → Import JSON → select the file

How often should you back up?

Recommended: once a month, or before changing phones. The file is small (a few KB) and can be saved anytime.

Where is the file saved on the phone?

iPhone: in the Files app → Downloads (or iCloud Drive, if you saved it there).

Android: in the Downloads folder.
